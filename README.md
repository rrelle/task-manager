# Task Manager App 📝

A simple and modern task management application built with Next.js and Tailwind CSS.

## 🚀 Features
- Add tasks
- Delete tasks
- Filter tasks (All, Active, Completed)
- Clean and responsive UI
- Gradient background with styled components

## 🛠️ Tech Stack
- Next.js
- React
- Tailwind CSS
- JavaScript
